## 📖 Overview

This repository documents the design, deployment and maintenance of my personal Home Lab infrastructure.

The project was created as a practical environment to strengthen my knowledge of Linux system administration, Docker containers, networking, DNS services and infrastructure monitoring.

Every deployed service is documented, tested and maintained following a structured workflow inspired by real-world IT environments.

## 🌐 Network Topology

The current Home Lab consists of a Debian 12 server connected to the local network through a TP-Link router.

The server provides DNS filtering, monitoring and self-hosted services using Docker containers.

A complete network topology diagram will be included in future updates.
